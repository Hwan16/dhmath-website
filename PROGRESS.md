### 이미지 / 아이콘 교체
- [ ] 헤더 로고 아이콘 교체 (현재: 📐 이모지 → 실제 로고 이미지)
- [ ] 메인 히어로 프로필 이미지 교체 (현재: 👩‍🏫 이모지 → 실제 선생님 사진)
- [ ] 메인 히어로 장식 아이콘 교체 (현재: ✏️, 📊 이모지 → 커스텀 아이콘/이미지)
- [ ] 파비콘 설정 (브라우저 탭 아이콘)
- [ ] Open Graph 이미지 (카톡/SNS 공유 시 미리보기 이미지)

> 📁 **이미지 파일 위치**: `public/images/` 폴더에 저장
> 📝 **교체 방법**: 이미지 준비 후 요청하면 코드 수정해드림

- [ ] Sentry 연동 (에러 자동 수집)
- [ ] Error Boundary 설정 (에러 발생 시 친절한 안내 화면)
- [ ] 에러 발생 시 이메일 알림 설정

> 🐛 **디버깅 방법**: 에러 발생 → Sentry 대시보드에서 에러 내용 복사 → AI에게 붙여넣기 → 해결책 받기
